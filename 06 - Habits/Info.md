# 🧘 Habits Folder

Folder ini digunakan untuk menyimpan semua file catatan kebiasaan individu Anda. 

Setiap berkas di dalam folder ini dibuat menggunakan templat **[Habit.md](file:///Users/sakhandaru/Documents/Obsidian%20Vault/99%20-%20Templates/Habit.md)** dan berisi grafik kontribusi (Contribution Graph) tersendiri untuk memantau kemajuan Anda.

### Cara Menambah Habit Baru:
1. Tap tombol **`🆕 New Habit`** di halaman **[Inbox.md](file:///Users/sakhandaru/Documents/Obsidian%20Vault/00%20-%20Home/Inbox.md)**.
2. Ikuti panduan pembuatan (Nama Habit & Frekuensi).
3. Tugas habit otomatis dijadwalkan dengan ikon 🌱 dan akan muncul secara berkala di halaman utama **[Today.md](file:///Users/sakhandaru/Documents/Obsidian%20Vault/00%20-%20Home/Today.md)**.
